\subsection{Descrição Detalhada}
\subsubsection*{template$<$typename T$>$class Pilha$<$ T $>$}

Classe \hyperlink{classPilha}{Pilha}. 

Uma pilha é uma lista linear na qual o primeiro elemento a entrar é o último elemento a sair. Ela possui apenas uma entrada onde os dados entram e de onde eles saem.
